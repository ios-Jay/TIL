# 연습문제 2: 랜덤 빙고 게임

> #### 조건문을 활용하여 스위프트로 랜덤 빙고 게임 만들기

- 내가 선택한 것과 컴퓨터가 랜덤으로 선택한 것을 비교하고 출력

```
var computerChoice: Int = Int.random(in: 1...10)

var myChoice: Int = Int.random(in: 1...10)



if computerChoice > myChoice {

    print("Up")

} else if computerChoice < myChoice {

    print("Down")

} else {

    print("Bingo")
}

```
